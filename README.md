# Ferry

## Usage
Clone the repository.
```$ git clone git@github.com:yasenne/ferry.git```
Move into ```ferry``` directory and create a virtual environment.
``````
$ cd ferry
$ python3 -m venv .venv
$ source .venv/bin/activate
``````
Initialize the database.
```$ flask --app ferry init-db```
Run the app.
```$ flask --app ferry run```
